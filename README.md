## How to use
Do the following to use this project: 
1. clone the project 
2. run npm install
3. to run tests: 
``
protractor protractor.conf.js
``

## How to install
* clone the repo; 
* run npm install in root folder; 

## How to run the tests
Run the command below to run the tests: 
``protractor protractor.conf.js``
